Do you like Atom? (yes/no):



1. Preview Html



Preview Html is a good tool because it allows for quick access to preview your code and make sure everything is working properly as you develop your web page.

You can quickly access a preview of you website by simply using using the command Ctr-Shift-H or you can right click and press "preview html"

2. Atom Beautify



Atom Beautify is a great tool for atom because it helps you to better organize your code and make it easier to read and to find things.

All you have to do is use the command Alt-Ctrl-B or simple right click and click "beautify editor contents"

3. Ask Stack



Ask Stack is another great tool because it helps you to answer questions without opening you browser. All you have to do is type in your question, and the coding language you are asking about (css, java, html, etc).

After that it searches the website stack overflow in the discussion forums for the question you ask. To use this tool you can simply use the command Ctrl-Alt-A or type in "Ask Stack" in your command bar.
